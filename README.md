# AI-HR-Flask
## Overview
This is a Flask-based HR Assistant that answers policy, leave & benefits queries.

## Features
- Chat interface
- Leave submission form
- HR statistics

## Tech Stack
- Flask
- HTML/CSS/JS
- Python

## Setup
1. Install Python 3.x
2. Install required packages: pip install -r requirements.txt
3. Run app: python app.py
4. Access at http://localhost:5000

## Potential Improvements
- Integrate AI model for smarter responses
- Add user authentication
